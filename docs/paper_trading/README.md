# Paper-Trading Validation Loop

This folder holds your day-by-day validation journal for Days 91-100.

Workflow (daily):
1. Run a screener snapshot (top N) into `docs/paper_trading/snapshots/`.
2. Create today's journal entry from the template.
3. Review results vs your manual judgment (false positives / false negatives).
4. Capture any changes you want to make later (do not tweak weights impulsively).
